# Pythoneala

A Python reference with syntax-highlighted code examples. Quick reference for Python developers covering data types, algorithms, data structures, and interview patterns.

**Live site:** [Pythoneala](https://neuronspeeed.github.io/python/)

## Features

- **Python Fundamentals** - Core syntax, statements, conditionals, loops
- **Data Types** - Strings, lists, dictionaries, sets, tuples
- **Functions & OOP** - Functions, classes, decorators, generators
- **Data Structures** - Arrays, linked lists, trees, graphs, heaps
- **Algorithms** - Sorting, searching, dynamic programming
- **Interview Prep** - Common patterns and problem-solving techniques
- **Big O Complexity** - Time and space complexity analysis

Each topic includes methods with complexity analysis and interactive code examples.

## Tech Stack

| Layer | Tech |
|-------|------|
| Framework | React 19 + TypeScript |
| Build | Vite |
| Styling | CSS variables |
| Deploy | GitHub Pages (via Actions) |

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/    # Reusable UI components
├── config/        # Routes and navigation
├── data/          # Method definitions by topic
├── pages/         # Page components
├── hooks/         # Custom React hooks
├── styles/        # CSS organized by component/page
└── utils/         # Utility functions
```
